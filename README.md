# IoT-Door
Introducing us to the wide world of IoT. Using IBM cloud, the project is a "interactive" mail system/door
DISCLAIMER; THIS PROJECT WORKS ONLY WITH DOORS, THAT HAVE A MAILSLOT.


Components used;
Arduino UNO
SPX1189-002 Infrared sensor
Pmod Oled RGB 96x64 LCD Screen


Goals of this project;
- Using different sensors, we are able to track if you recived mail. We are using a SPX1189-002 Infrared sensor for this.
- LCD Screen shows if you recived mail (OPTIONAL; LCD screen shows if you are home or not).
- Using IBM Cloud IoT platform, you will recive a text message, if you got mail.
